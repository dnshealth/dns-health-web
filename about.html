About

DNSHealth is a tool designed to verify the compliance of a DNS server and a DNS zone contained within to the specifications specified in the IANA nameserver technical standard. The tool will check if the zone for a domain name has been correctly set up at the specified nameservers. Based on this information, the user can determine if a domain is ready to be delegated to those nameservers. The tool is planned to have a web interface for anyone to use as well as an REST API.


It's very useful if you have purchased a domain name and wish to take it into use by pointing it to nameservers. Using the tool you can see if the nameservers are working correctly.


This is a project from KTH to make a DNSchecker for NordName. NordName plans to use the tool to check the technical compliance of the nameservers when a customer wants to change the nameservers on a domain. The command line tool can already be used for this purpose as it can output results in JSON format, which can be read by another program.
